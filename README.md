**Bamboo Engine** is a self-developed lightweight simulation engine that supports high-fidelity graphics rendering and various AI simulation algorithms.

## Dependencies
- vulkan
- vma
- glm
- glfw
- imgui
- spdlog
- json
- yaml-cpp
- tinygltf
- cereal
- rttr
- eventpp
